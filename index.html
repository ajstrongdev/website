Hello, I'm AJ! This is my little home on the internet where I like to show off things I have created! I am a GNU/Linux enthusiast and founder of Rhino Linux and The Orbital Desktop Environment.

I'm a software developer and information activist, I seek to promote accessible, reliable information about free and open source software in my blog posts and write-ups. I am 18 years old, and I am very passionate about computers and technology.

I first started learning website development in 2017 from school and fell in love with programming, from there I began to teach myself as much as I could. I was introduced to GNU/Linux and open source software in August 2020 and swapped from Windows 10 to Linux full-time in October 2020. I fell in love with open source software and the philosophy behind it.

My Projects

Rhino Linux - I am the founder and current Desktop Lead of Rhino Linux, the successor distribution to my other project, Rolling Rhino Remix. Rhino Linux is a rolling release Ubuntu-based distribution with Pacstall & XFCE at its core.
• I was also formerly Project Lead of Rhino Linux until the role was dissolved in 2023.

PyLure - PyLure is a simple, intuitive and easy to use Python text editor. It utilises TkInter for a truly cross-platform approach, that will work on macOS, Linux and Windows.

Orbital Desktop - Orbital is a lightweight desktop environment that provides tiling window management with class and style. Orbital is powered by skyWM, a minimalistic, extensible window manager written in Rust.

skyWM - skyWM is a minimalistic window manager written in Rust. skyWM takes inspiration from the Unix philosophy of doing one thing and doing it well. skyWM is now utilised in the Orbital Desktop Environment.

HepLang - HepLang is a programming language I created as a hobby project in Python. It takes inspiration from the Rust syntax and is not yet feature complete.
